### -NumGame5.py
游戏规则如下：<br>
主持人确定猜数字的范围和pk次数<br>
每位参赛者按顺序输入自己的名字<br>
主持人确定每次可以猜的次数（默认4次）<br>
开始比赛-每位参赛者按顺序猜一次，如果猜对加一分<br>
如果在规定次数大家都没有猜出来，打印出答案<br>
比赛结束后打印排行榜<br>
